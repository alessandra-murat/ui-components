# Botões


<Canvas of={ButtonStories.AllSizes} />

Os componentes de botões são destinados para os que os usuários realizem ações específicas através de um clique. Com suporte para diferentes variantes e tamanhos, permitindo a inserção de textos, ícones ou ambos.


### Aplicação com Texto

```jsx
<Button.Root>
  <Button.Text text="Button" />
</Button.Root>
```

<Canvas of={ButtonStories.Default} />

### Aplicação com Texto e Ícone à Esquerda

```jsx

  <Button.Text text="Button" />
</Button.Root>
```

### Propriedades

- Variants: **filled**, **outlined** e **borderless**;
- Size: **sm**, **md** e **lg**;
- Disabled: **true** e **false**;
- Shape: **square** e **rounded**
